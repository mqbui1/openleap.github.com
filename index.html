OpenLeap

OpenLeap · Open-source programs for the Leap

The Leap is an incredible new motion-tracking device from Leap Motion. It tracks movements up to 10 micrometers (!) in 8 cubic feet of wonderful, beautiful, magnificent interactivity. OpenLeap is a community of open-source enthusiasts making software for this wonderful device.

See the GitHub orginization

Who?

Started by Owen Versteeg as an open-source initiative for the Leap. Comment on the LM dev question and I'll add you to the org!

What

A bunch of open-source applications that use the Leap. One example is DrawPad.

Ooh! I have an idea for an app!

Awesome! Create a repo if you can code, or email me if you can't.

Why?

Many Leap developers asked for a good community site where they could share projects and see examples - so here one is!

How do I add a repo?

Just comment on the LM question thread with your repo and I'll add it!

I don't have a Leap!
